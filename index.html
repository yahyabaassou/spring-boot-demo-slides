What's Spring Boot?

Spring Boot makes it easy to create Spring-powered, production-grade applications and services with absolute minimum fuss. It takes an opinionated view of the Spring platform so that new and existing users can quickly get to the bits they need.

Spring Boot initializr

To boostrap your application with Spring Initilizr, you have to fill in few informations:

  • Project type: Maven or Gradle
  • Language: Java, Kotlin or Groovy
  • Spring Boot version
  • Project Metadata (group, artifact)
  • Additional dependencies

You can also be more precise by fill in also packaging mode (jar or war) and java version.

You could achieve the same result with a simple curl command 

							
								// This command generates a spring-boot-demo directory
								// holding a Maven web-based Java project:
								$ curl https://start.spring.io/starter.tgz \
										-d dependencies=web \
										-d language=java \
										-d javaVersion=1.8 \
										-d type=maven-project \
										-d baseDir=spring-boot-demo | tar -xzvf -
